
# Social Media App

Bu projede [JSONPlaceholder](http://jsonplaceholder.typicode.com/)'a ait API kullanılarak burdaki verilere göre basic tarzda bir uygulama oluşturuldu.

## Tech Stack
- Jetpack Compose
- MVVM
- Coroutines
- Dagger-Hilt
- Paging3
- Navigation
- Flow
- Coil
